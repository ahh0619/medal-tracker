# Olympic Medal Tracker 만들기 
## ✨ 목표 정하기
- **웹 개발** 전반의 **흐름**과 **팀과의 작업**이 어떤 흐름으로 흘러가는지 이해할 수 있습니다.
- **Modal**을 이용하여 팀원들의 정보를 화면에 구현하였습니다.
- **오픈API**를 활용해 사용자의 **날짜 및 날씨정보**를 구현할 수 있습니다.
- **CRUD**를 활용하고 구현할 수 있습니다.
